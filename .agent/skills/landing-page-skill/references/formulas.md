# Landing Page Formulas

Use these formulas to draft high-converting copy.

## 1. The Hook (Hero Header)
**Formula:** `[End transformation result] + [How it works]`
- *Example:* "Get 10x more leads [Transformation] with our AI-powered outreach bot [How it works]."
- *Example:* "Launch your startup in a weekend [Transformation] using our no-code template library [How it works]."

## 2. The Context (Subheader)
**Purpose:** Address the biggest objection or explain the "Secret Sauce."
- *Formula:* "The only [Category] that [Solves major pain point] without [Common friction]."
- *Example:* "The only CRM that syncs your emails automatically without manual data entry."

## 3. The Instruction (CTA)
**Rule:** Action-oriented and low-friction.
- *Good:* "Start Free Trial," "Get My Custom Plan," "See it in Action."
- *Bad:* "Submit," "Sign Up," "Learn More."

## 4. Value Propositions
**Formula:** `[Result/Benefit] + [Concrete Feature]`
- *Example:* "Stop wasting time on admin [Benefit] with automated invoice generation [Feature]."

## 5. Objection Handling
**Mechanism:** Identify → Validate → Solve.
- *Example:* "Worried about setup? [Identify] We handle the migration for you [Solve], and you'll be live in under 10 minutes [Validate specificity]."
