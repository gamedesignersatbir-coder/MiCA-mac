# Skill Creator

A meta-skill for creating and packaging other skills. This skill provides tools and guidance for building high-quality, reusable agent skills.

## Overview

Creating a skill involves:
1. Defining the skill's purpose and system prompt
2. Implementing the core logic (usually in Python)
3. Defining tools and their schemas
4. Packaging and validating the skill

## Features

- **Interactive Initialization**: Guided process to set up a new skill directory.
- **Validation**: Automated checks to ensure your skill follows best practices.
- **Packaging**: Tools to bundle your skill for distribution.
